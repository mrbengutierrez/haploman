Brief Description

This project was done while I was an undergraduate researcher at MIT, and eventually became the content of my thesis. The robot, termed "Manus Home" is an in home robotic solution to stroke upper limb rehabilitation. The robot was designed to optimize the cost of manufacturing while optimizing portability and precision.
