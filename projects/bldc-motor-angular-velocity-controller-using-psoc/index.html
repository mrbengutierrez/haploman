Brief Description

This project was completed as part of a final project for my power electronics course at MIT. This was my first closed looped controlled BLDC motor controller that I had ever made. The motor controller used the PSOC (Programmable System On Chip), as the logic hardware. The PSOC was chosen because it allowed me to test circuits faster than by building them myself. The PSOC allows for reconfigurable analog and digital circuits to be programmed.

In order to control the angular velocity of the motor, I used a simple proportional controller. The proportional controller would simply increase the voltage going into the BLDC motor if an external torque was applied in order to keep the desired angular velocity constant. Conversely, if the external torque on the motor decreased, the controller would compensate by decreasing the voltage into the motor.
